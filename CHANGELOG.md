## [0.1.0] - 2026-01-06

### Added
- Initial release of BangunanPro
- Product management with categories and suppliers
- Sales and POS system with discount support
- Customer management (retail, wholesale, contractor)
- Supplier management
- Real-time inventory tracking
- Stock movement history
- Price history tracking
- Audit logging for sensitive actions
- Role-based access control (Admin, Manager, Cashier)
- Multi-user support
- Dashboard with analytics
- Export to PDF receipts
- 646 pre-loaded Indonesian hardware products
- Livewire reactive components
- Responsive mobile interface

### Technical
- Laravel 12.x framework
- Livewire 3.x for frontend
- TailwindCSS 3.x for styling
- MySQL 8.x database
- Service layer architecture
- Repository pattern
- Event-driven architecture
- Observer pattern for model events
